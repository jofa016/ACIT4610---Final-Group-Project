# Problem 5 – Reinforcement Learning: FrozenLake-v1

### Environment
- Gymnasium `FrozenLake-v1`, 6x6 map, `is_slippery=True`.

### How to Run
1. `pip install -r requirements.txt`
2. Open `notebooks/frozenlake_qlearning.ipynb`
3. Run all cells to train and evaluate the agent.

### Files
- `src/qlearning_agent.py` – core Q-learning implementation
- `results/` – saved training curves and success-rate plots

### Libraries
Gymnasium, NumPy, Matplotlib
